To generate Pickel Files:<br>
Copy the code of PickleFile.py and input the Loan_Risk_Data.xlsx

Make sure Pickle files (encodes.pkl and loan_Model.pkl) are in the same folder structure as app.py<br>
_.venv<br>
|_app.py<br>
|_encoders.pkl<br>
|_loan_models.pkl<br>

Steps to deploy the Project:<br>
 pip install streamlit pandas scikit-learn<br>
 streamlit run app.py
